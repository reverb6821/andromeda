# Header

Header component with different props.

#### Header

<Canvas of={HeaderStory.HeaderExample} />

## Usage

```ts
import Header from '@/components/ui/navigation/Header';

const Header = () => {
  return (
        <Header headerTitle='Andromeda' headerEl={headerEl}/>
  );
};

export default Header;
```

#### Arguments
- **headerLogo** - A string that represents the img source.
- **headerTitle** - A string that reppresent the Title
- **headerSubTitle** - A string that reppresent the subtiitle (it is optional)
- **headerEl** - An array of object that represent the element inside the menu.. This object are label and routes.
