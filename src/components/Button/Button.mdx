# Button

Button component with different props.

#### Button

<Canvas of={ButtonStory.Primary} />

## Usage

```ts
import Button from '@/components/button/Button.tsx;

const Button = () => {
  return (
      <Button
        onBtnClick={(event: React.MouseEvent) => {
            alert('You clicked that button!');
        }}
        btnStyle='primary'
        type="submit"
        label='Btn Example'
        disabled=false
        isLoading=false
      />
  );
};

export default Button;
```

#### Arguments
- **label** - A string that represents the text content of the button.
- **btnStyle** -  A string indicating whether the button should have a primary styling or not. Typically, a primary button stands out as the main action in a user interface. It accept one of following value: one of the possible values:
  - "primary",
  - "primary--alert",
  - "primary--warn",
  - "primary--success".
  - "secondary",
  - "secondary--alert",
  - "secondary--warn",
  - "secondary--success",
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **isLoading** - A boolean indicating whether the button should be disabled or not during an action is running. When disabled, the button cannot be clicked or interacted with.
- **icon** - A string indicating whether the button should have a icon. It accept remix icon font class
- **onBtnClick** _`() => void`_ - A function that is called when the button is clicked. It receives a MouseEventHandler for handling the click event on the button element.
