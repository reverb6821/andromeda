# Badge

Badge component with different props.

<Canvas of={BadgeStory.Primary} />

## Usage

```ts
import Badge from '@/components/partials/Badge';

const Badge = () => {
  return (
      <Badge
            primary
            label= 'Label Example'
        />
  );
};

export default Button;
```

#### Arguments
- **label** - A string that represents the text content of the button.
- **badgeStyle** -  A string indicating whether the button should have a primary styling or not. Typically, a primary button stands out as the main action in a user interface. It accept one of following value: one of the possible values:
  - "primary",
  - "primary--alert",
  - "primary--warn",
  - "primary--success".
  - "secondary",
  - "secondary--alert",
  - "secondary--warn",
  - "secondary--success",
- **icon** - A string indicating whether the button should have a icon. It accept remix icon font class
