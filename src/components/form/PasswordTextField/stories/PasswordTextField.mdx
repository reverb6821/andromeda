# Password Text Field

Password Text Field component with different props.

<Canvas of={PasswordTextFieldStory.PasswordTextFieldExample} />

## Usage

```ts
import PasswordField from '@/components/form/PasswordField/PasswordField';

const PasswordField = () => {
  return (
      <PasswordField
        label= 'Password',
        model= 'login.password'

        />
  );
};

export default PasswordField;
```
#### Arguments
- **label** - A string that represents the text content of the field.
- **disabled** - A boolean indicating whether the button should be disabled or not. When disabled, the button cannot be clicked or interacted with.
- **helper** - A string that represents the description of the content field.
- **placeholder** - A string that represents the text content of the field.
- **model** - A string that represents the field name inside formik form.
- **value** - Represents the field value.
