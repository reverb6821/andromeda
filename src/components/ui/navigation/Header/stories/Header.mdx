## Usage

```ts
import Header from '@/components/ui/navigation/Header';
import Logo from '@/assets/logo-placeholder-image.png'

 const headerEl: [
    {label: 'Home', route: ''},
    {label: 'About', route: '/about'},
    {label: 'Contact', route: '/contact'}
  ]

const Example = () => {
  return (
        <Header
          headerLogo={Logo}
          headerTitle='Andromeda'
          headerSubTitle='A component Collection'
          headerEl={headerEl}
        />
  );
};

export default Example;
```

#### Arguments
- **headerLogo** - A string that represents the img source.
- **headerTitle** - A string that reppresent the Title
- **headerSubTitle** - A string that reppresent the subtitle (it is optional)
- **headerEl** - An array of object that represent the element inside the menu. The Object should be as follow:
  - label: Name of the Route.
  - route: The path (ex. '/home')
