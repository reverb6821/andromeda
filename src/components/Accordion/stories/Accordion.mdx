# Accordion

Description for Accordion component.

<Canvas of={ AccordionStory.Default } />

## Usage

```ts
import Accordion from '@/components/Accordion/Accordion;

const Example = () => {
  return (
      <Accordion
        title='Accordion Example'
        message='Lorem Ipsum Dolor Sit Amet'
      />
  );
};

export default Example;
```

#### Arguments
- **title** - Indicate the main title of the element
- **subtitle** - Indicate the subtitle of the element (if necessary)
- **icon** - Indicate the icon for the main title of the element (if necessary)
- **message** - Indicate the main text of the element that show on clicks
